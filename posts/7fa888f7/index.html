2020年终总结

2021-02-03
字数统计: 1.2k   |   阅读时长≈ 3 分钟

2020 庚子年，注定是不平凡的一年。

一场突如其来的疫情打乱了所有人的生活节奏，也打破了我对生活的幻想，让我从一个身在异乡的游子产生了落叶归根的想法。「家乡」是所有游子心底的港湾，「回家」是所有人都绕不开的话题。作为一个土生土长的陕西人，西安是故土，西安是家乡，西安是摇篮。作为一个高速发展的中西部省会城市，西安是明珠，西安是奇迹，西安是未来。

有了想法，就应该去行动。我收拾好自己带上满满的憧憬来到了西安这座神往已久的城市，开始了我的回家之旅。

工作是生活的保障，转眼间我已经在西安生活了两个多月，也在贵公司工作两个多月。

初入公司，融入公司乃第一要务。

作为一名技术人员，技术栈的融合至关重要。通过完成每天既定的工作任务，渐渐的将自己的技术栈与公司统一起来。在不长的时间里顺利的完成了以下技术栈的储备:

  1. 熟练掌握了使用 Vue 框架
  2. 熟练使用 IDEA 调试 Java 项目
  3. 熟练使用 Thymeleaf 框架完成 Web 开发

并且结合自身的技术栈圆满的完成了以下工作任务:

  1. 后台管理平台开发。

    在单一窗口平台建设和物品流转信息平台建设中，后台管理界面的建设我都参与到了其中。从页面布局到接口对接，从功能完善到错误修复，我都尽我所能的发挥了作用。

  2. 公众号 H5 开发。

    在物品流转信息平台项目中，用户使用微信公众号入口调用微信扫一扫功能获取对应报关单号的流转信息。

  3. 综合服务平台前端页面开发。

    现有的产品随着时间的推移，其技术架构陈旧、安全漏洞增多、可维护性极差，只有将其推倒重建。

    我使用官方推荐的、最新的前端开发框架，加上模块化、组件化的思想重构了此项目。从而大大提高了安全性和可维护性。

  4. OAuth2.0 调研和区块链项目调研。

    任何技术解决方案，使用的前提就是了解。通过既定的业务场景和自身的经验储备去判断众多解决方案的优劣才能挑选出合适的项目来使用。

    了解项目的同时又丰富了自身的经验储备。

作为一名职业人，融入团队才是立根之本。一个人的力量能力再大，还是不能和一个团队相比的。每个人在团队中找到自己的位置，处理好团队之间的关系，才能充分的发挥自己的能力。

在技术选型和难关攻克时同事的点拨有如醍醐灌顶之功效；在技术栈的融合过程中及时的请教他人达到了事半功倍的效果；在我擅长的领域里我也能耐心的教导给他人。总之目前所处的是一个很棒的团队，只有这么优秀的团队才能让一个临时发起的项目在短短一周内完成开发、测试、上线及运行。

作为一名员工，完成工作任务是职责，提升工作技能是职责所需。想要提升自己就需要客观的分析评价自己，才能为日后的发展找到正确的方向。经过最近几个月的工作发现需要在以下几个方面弥补自己的不足：

  1. 丰富理论知识，用理论指导编码，用代码阐释理论。从而改善长期以来赶进度、赶功能形成的只明白如何使用，不理解背后的原理的错误习惯，提高代码质量和工作产出。

  2. 学习从不同角度、不同维度看待、理解项目。从而开发出更完善的产品。

  3. 学习使用业务流程图表等工具。从而更规范、方便、流畅的与他人沟通项目。

  4. 学习自动化部署及容器化解决方案，提高工作效率。

  5. 深入了解区块链项目，理解其商业应用场景及解决方案。

有了目标，就需要去实现。我希望在新的一年里能够瞄准目标披荆斩浪，努力前行。

最后，希望大家都能在牛年借着牛气一飞冲天。
